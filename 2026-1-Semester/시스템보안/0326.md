  - Virtual Memory Concept(29.p)
    - Separation of logical memory(논리 주소) from physical memory(물리 주소)
      - Logical address space(논리적 주소 공간)이 물리적 메모리 크기를 초과할 수 있음

    - Benefits
      - 다수의 프로세스가 주소 공간을 공유함
      - 더 효율적인 프로세스 생성
      - 더 많은 프로그램들이 동시에 실행 가능함

  - Logical vs Physical Address
    - Logical address(Virtual address)
      - 실행 중 CPU에 의해 생성됨
      - 각 프로세스는 자신만의 논리적 주소 공간을 가짐
     
    - Physical address
      - RAM(물리적 메모리)
