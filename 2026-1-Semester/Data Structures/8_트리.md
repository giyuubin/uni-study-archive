  - 선형(linear) 자료 구조: 리스트, 스택, 큐
  - 계층적인(hierarchical) 구조: **트리**
  - 트리의 정의: 부모-자식 관계의 노드들로 이루어짐
  - 응용 분야: 계층적인 조직 표현, 컴퓨터 디스크의 디렉토리 구조, 인공지능에서의 결정 트리

  - 트리의 종류:
    - 일반트리
    - **이진트리(binary tree)**
      - 모든 노드가 2개의 서브 트리를 가지고 있는 트리
      - <img width="357" height="193" alt="image" src="https://github.com/user-attachments/assets/d1e32c41-130f-49a1-a910-ba92a23b8b4c" />

      - 서브 트리는 공집합일 수 있음
      - 이진 트리의 노드에는 최대 2개까지의 자식 노드가 존재
      - 모든 노드의 차수는 2개 이하가 된다 // 관리가 편함
      - 서브 트리간의 순서가 존재(왼쪽 서브 트리, 오른쪽 서브 트리)

        - 포화 이진트리
          - 각 레벨에 노드가 꽉 차있는 이진트리
          - 높이 k인 포화 이진트리는 $2^k-1$개의 노드
          - 노드에 번호 부여하는 방법:
            - 레벨 단위로 왼쪽에서 오른쪽 순서로 부여
        - 완전 이진트리
          - 높이가 k일때 레벨 1~k-1까지는 노드가 모두 채워져 있음
          - 마지막 레벨 k에서는 왼쪽부터 오른쪽으로 노드가 순서대로 채워져 있음
            - 마지막 레벨에서는 노드가 꽉차있지 않아도 되지만 중간에 빈곳이 있으면 안됨

      - 이진트리 표현(pp. 12~13)

          - 연결 리스트와 유사한 구조
            - 구조체를 이용하여 노드의 구조 정의

      - 이진트리 순회
        - **순회**: 이진 트리의 노드들을 방문하여 노드가 가지고 있는 데이터를 목적에 맞게 처리하는 연산
        - 순환 기법:
          - 전체 트리와 서브 트리의 구조가 동일함
          - 전체 트리의 사용된 알고리즘을 서브 트리에 적용(순환 호출) => 문제의 크기가 작아짐
          - 문제의 구조는 같고, 크기만 작아지는 경우 순환이 효과적임
          - 기본적인 순회 방법 // 그림 한 번 보자(pp. 17~)

      - 트리 응용 // 코드 넘기심
        - 수식트리
          - 산술식을 트리형태로 표현한 것
            - 비단말노드: 연산자
            - 단말노드: 피연산자
          - <img width="445" height="229" alt="image" src="https://github.com/user-attachments/assets/d0e6d42a-48ad-4c59-a795-c75b8e8537f3" />

          - 후위순회 사용
            - 후위 표기식으로 계산하기 위함
            - 루트보다 자식 노드들을 먼저 방문하는 방법
              - 서브트리의 값을 순환호출로 계산
              - 비단말노드의 양쪽 서브트리의 값을 노드에 저장된 연산자를 이용하여 계산
                 
          - 노드 개수
            - 탐색 트리안의 노드 개수를 계산
            - 각각의 서브트리에 대하여 순환 호출한 다음, 반환되는 값에 1을 더하여 반환
            - <img width="689" height="177" alt="image" src="https://github.com/user-attachments/assets/3a7624e0-c967-43c7-9428-283cba0b4d67" />

          - 단말 노드 개수
            - 트리의 모든 노드를 순회
              - 왼쪽 자식과 오른쪽 자식이 동시에 0이면 단말노드이며 1을 반환
              - 비단말 노드일 경우 각각의 서브트리에 대하여 순환 호출 후, 반환되는 값을 서로 더한다
