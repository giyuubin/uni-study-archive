### 6. 사이버 공격은 크게 수동적 공격(passive attack)과 적극적 공격(active attack)으로 나뉜다. 각 공격의 의미를 쓰고, 예를 하나 이상 들어보시오.

  - 수동적 공격 (Passive Attack): 시스템 자원에 직접적인 영향을 미치거나 변경을 가하지 않으면서, 그저 시스템으로부터 정보를 획득하거나 전송되는 메시지를 도청하는 형태의 공격이다.
    - 예시: 메시지 내용 도청(Eavesdropping), 트래픽 분석(Traffic analysis)
  - 적극적 공격 (Active Attack): 데이터 스트림을 수정하거나 허위 데이터를 생성하여 시스템 자원의 작동에 고의적으로 영향을 미치고 피해를 주는 공격이다.
    - 예시: 신분 위장(Masquerade), 데이터 재전송(Replay), 메시지 내용 수정, 서비스 거부(Denial of service)

### 8. 베쥬 항등식의 성질(두 정수 $a, b$에 대해 $gcd(a,b) = ax+by$인 $x, y$가 존재)을 이용해서 $Z_p$(단, $p$는 소수)상의 어떤 임의의 정수의 곱셈 (mod $p$)에 대한 역원을 계산할 수 있음을 설명하시오.

- 베쥬 항등식에 따르면 두 정수 $a, p$의 최대공약수가 $gcd(a,p)$일 때 $ax+py=gcd(a,p)$를 만족하는 정수 $x,y$가 존재한다.
- 만약 $p$가 소수이고, $a$가 $p$의 배수가 아니라면, 두 수는 서로소이므로 $gcd(a,p)=1$이 된다. 따라서 식은 $ax+py=1$이 되며, 양변에 (mod $p$) 연산을 취하면 $pq$는 $p$의 배수이므로 0이 되어 $ax \equiv 1 \pmod p$가 성립한다.
- $\therefore$ 이 식을 만족하는 $x$를 구하는 것이 곧 $a$ 모듈러 $p$에 대한 곱셈의 역원을 구하는 것과 동일함을 알 수 있다.

### 10. $GF(2^8)/[x^8+x^4+x^3+x+1]$에서 $x^7+x+1$의 곱셈에 대한 역원이 $x^7$임을 증명하시오.

- 두 다항식을 곱하여 모듈러 $x^8+x^4+x^3+x+1$ 연산을 수행한 결과가 1이 나오는지 확인.
- 1. $(x^7+x+1) \times x^7 = x^{14}+x^8+x^7$
- 2. 모듈러 다항식 $m(x) = x^8+x^4+x^3+x+1이므로, x^8 \equiv x^4+x^3+x+1$로 치환할 수 있다.
- 3. 차수를 낮추기 위해 $x*{14}$를 $x^8$로 나눈다:
  - $x^{14} = x^6 \times x^8 = x^6(x^4 + x^3 + x + 1) = x^{10} + x^9 + x^7 + x^6$
  - $x^{10} = x^2 \times x^8 = x^2(x^4 + x^3 + x + 1) = x^6 + x^5 + x^3 + x^2$
  - $x^9 = x \times x^8 = x(x^4 + x^3 + x + 1) = x^5 + x^4 + x^2 + x$
  - 따라서 $x^{14} = (x^6 + x^5 + x^3 + x^2) + (x^5 + x^4 + x^2 + x) + x^7 + x^6 = x^7 + x^4 + x^3 + x$
- 4. 모든 항을 원래 식에 대입하여 합산한다:
  - $x^{14} + x^8 + x^7 = (x^7 + x^4 + x^3 + x) + (x^4 + x^3 + x + 1) + x^7$
  - 같은 차수의 항들은 XOR 연산하면 모두 상쇄되고 최종적으로 1만 남는다.
- 5. 곱한 결과가 1이므로 $x^7$은 $x^7 + x + 1$의 역원이 맞다.
