# 인공지능개론 6주차 학습 노트: 인공신경망(Neural Networks)

## 1. 생물학적 신경망과 인공신경망의 기원
인공신경망(Artificial Neural Network, ANN)은 인간의 뇌를 구성하는 생물학적 신경망의 구조와 동작 원리에서 영감을 받아 만들어진 수학적 학습 모델입니다.

* **생물학적 뉴런의 구조와 동작:**
    * **수상돌기(Dendrite):** 다른 뉴런들로부터 전기적 신호(입력)를 수신합니다.
    * **세포체(Cell body) & 축삭둔덕(Axon hillock):** 입력된 신호들을 모아 처리하며, 특정 임계치를 넘으면 활성화되어 신호를 발생시킵니다.
    * **축삭돌기(Axon) & 시냅스 말단(Synaptic terminals):** 발생한 신호를 다음 뉴런으로 전달합니다.
* **역사적 배경:**
    * 1943년, McCulloch & Pitts: 신경 활동에 내재된 논리를 수학적으로 계산하는 모델 제안.
    * 1958년, Frank Rosenblatt: **퍼셉트론(Perceptron)**이라는 초기 인공신경망 모델 고안. (입력, 가중치, 합산, 활성화 함수로 이루어진 단순 구조)
    * 1998년: 합성곱 신경망(CNN)의 초기 형태인 LeNet-5가 등장하여 이미지의 특징을 추출하고 처리하는 아키텍처 제시.

*(참고: 교안 1p ~ 11p)*

## 2. 퍼셉트론(Perceptron)과 활성화 함수(Activation Function)
인공신경망은 네트워크의 구조와 파라미터(가중치)를 기반으로 입력(Inputs)에서 출력(Outputs)으로 가는 수학적 함수를 모델링합니다.

### 2.1 인공 뉴런의 수학적 모델링
하나의 뉴런이 입력값 $x_1, x_2$를 받을 때, 각 입력의 중요도를 조절하는 가중치(Weight) $w_1, w_2$가 곱해집니다. 여기에 편향(Bias) $w_0$가 더해져 선형 결합(Net input function)을 이룹니다. 이 값은 그래프상에서 하나의 직선(경계)을 의미합니다.
$$Net\ Input = w_0 + w_1 x_1 + w_2 x_2$$

이후 이 값은 **활성화 함수(Activation Function)** $g(x)$를 통과하여 최종 출력(Output)을 결정합니다.
$$h(x_1, x_2) = g(w_0 + w_1 x_1 + w_2 x_2)$$

### 2.2 대표적인 활성화 함수
입력된 총합을 출력으로 어떻게 변환할지 결정하는 함수입니다. 교안의 그래프와 특징은 다음과 같습니다.

1.  **Step Function (계단 함수)**
    * **수식:** $g(x) = 1 \text{ if } x \ge 0, \text{ else } 0$
    * **특징:** 입력합이 0을 넘으면 1, 아니면 0을 출력하는 직관적인 형태입니다. 그러나 미분이 불가능하여 현대의 신경망 학습(성능 향상)에는 잘 쓰이지 않습니다.
2.  **Logistic Sigmoid (시그모이드 함수)**
    * **수식:** $g(x) = \frac{e^x}{e^x + 1}$
    * **특징:** S자 형태의 부드러운 곡선으로, 출력값을 0과 1 사이의 확률값처럼 매핑합니다. 하지만 자연상수 $e$ 연산이 들어가 계산이 느리다는 단점이 있습니다.
3.  **Rectified Linear Unit (ReLU)**
    * **수식:** $g(x) = \max(0, x)$
    * **특징:** 0 이하의 값은 0으로, 0 초과의 값은 그대로 출력하는 꺾인 직선 형태입니다. 연산이 매우 빠르고 성능이 압도적으로 좋아 현대 딥러닝에서 가장 널리 쓰입니다(교안 내 'Goat', '빨라짐' 표기).

*(참고: 교안 12p ~ 20p)*

## 3. 논리 연산 학습과 퍼셉트론의 한계
단층 퍼셉트론은 가중치를 조정해 나가며 주어진 입력에 대해 원하는 출력을 내도록 '학습'할 수 있습니다. 

### 3.1 OR / AND 연산의 분류 (선형 분리)
* **OR 연산 예시:** 입력 $(x_1, x_2)$ 쌍이 $(0,0)$일 때만 $0$을 출력하고, 나머지는 $1$을 출력해야 합니다.
    * 가중치가 $w_0 = -1, w_1 = 1, w_2 = 1$일 때: $g(-1 + 1 \cdot x_1 + 1 \cdot x_2)$
    * 입력이 $(0,0)$이면 $g(-1) = 0$. 입력이 $(1,0)$이면 $g(0) = 1$ (Step 함수 기준). 이처럼 알맞은 가중치를 통해 데이터를 성공적으로 분류합니다.
* **AND 연산 예시:** 가중치를 $w_0 = -2, w_1 = 1, w_2 = 1$로 조정하여 $(1,1)$일 때만 1이 나오도록 직선의 위치를 이동시킵니다.

### 3.2 선형 분리의 한계 (XOR 문제)
* 교안에 등장하는 빨간 점과 파란 점이 섞인 그래프는 퍼셉트론의 본질적인 한계를 보여줍니다.
* 퍼셉트론은 **선형 분리가 가능한 결정 경계(Linearly separable decision boundary)**, 즉 단 하나의 '직선'만 그을 수 있습니다.
* 따라서 대각선 방향으로 교차 배치된 **XOR 문제**는 하나의 직선으로 빨간 점과 파란 점을 나눌 수 없으므로, 단층 퍼셉트론으로는 절대 풀 수 없습니다.

*(참고: 교안 21p ~ 36p, 47p ~ 61p)*

## 4. 손실 함수와 경사 하강법 (Gradient Descent)
다수의 노드와 여러 선택지가 있는 모델(예: 날씨 예측 - 비, 맑음, 흐림, 눈)에서 가장 확률이 높은 값(예: 맑음 0.6)을 도출하기 위해서는 입력에 곱해지는 '가중치(Weight)'를 지속적으로 업데이트해야 합니다.

* **손실 함수(Loss Function/Error):** 모델의 예측값과 실제 정답 사이의 오차(Error)를 수치화한 함수입니다. 신경망 훈련의 최종 목표는 이 Loss를 최소화하는 것입니다.
* **경사 하강법 (Gradient Descent):** 1. 무작위 가중치로 시작합니다.
    2. 모든 데이터 포인트를 기반으로 오차를 줄이는 방향(기울기, Gradient)을 계산합니다. (교안의 2차원 그릇 모양 그래프에서 가장 낮은 골짜기를 향해 화살표 방향으로 내려가는 원리)
    3. 기울기에 따라 가중치를 업데이트합니다.

### 4.1 경사 하강법의 종류
| 종류 | 설명 | 특징 |
| :--- | :--- | :--- |
| **Gradient Descent** | **전체 데이터(All data)**를 한 번에 사용하여 기울기를 계산. | 계산이 정확하지만 속도가 매우 느림. |
| **Stochastic Gradient Descent (SGD)** | **단 1개의 데이터 포인트(One data point)**를 무작위로 뽑아 기울기를 계산. | 속도가 빠르지만 방향이 불안정하여 엉뚱한 곳(Local minima)에 빠질 가능성이 있음. |
| **Mini-Batch Gradient Descent** | 전체 데이터를 작게 쪼갠 **미니 배치(Small batch)** 단위로 기울기를 계산. | SGD의 불안정성과 전체 GD의 느린 속도를 절충한 현대의 표준 방식. |

*(참고: 교안 37p ~ 46p)*

## 5. 다층 신경망(MLP)과 역전파(Backpropagation)
단층 퍼셉트론의 XOR 문제를 해결하기 위해 도입된 것이 다층 신경망입니다.

* **다층 신경망 (Multilayer Neural Network):** 입력층(Input Layer)과 출력층(Output Layer) 사이에 최소 하나 이상의 **은닉층(Hidden Layer)**을 가진 신경망입니다.
* **비선형 결정 경계:** 은닉층(블랙박스 역할을 함)이 추가되면서, 모델은 단순한 직선이 아닌 꺾이고 휘어진 복잡한 결정 경계(Decision boundary)를 만들어 데이터를 분류할 수 있게 됩니다.

### 5.1 역전파 알고리즘 (Backpropagation)
은닉층이 있는 모델은 출력층에서 발생한 오차를 앞쪽(은닉층) 가중치에 어떻게 배분할지가 문제입니다.
1. 순전파(Feed-forward)를 통해 최종 예측값을 내고 **출력층의 오차(Error)를 계산**합니다.
2. 출력층부터 시작하여 가장 초기 은닉층 방향으로 **역방향(Inwards)으로 이동하며 오차를 전파(Propagate error back)** 합니다. (출력에서 난 에러를 다시 뒤로 보냄)
3. 각 층에 전파된 오차의 기여도를 바탕으로 **각 노드의 가중치를 업데이트**합니다.

*(참고: 교안 62p ~ 78p)*

## 6. 심층 신경망(DNN)과 과적합 문제
* **심층 신경망 (Deep Neural Networks, 딥러닝):** 은닉층(Hidden Layer)이 여러 개(Multiple) 쌓여 깊어진 신경망 구조를 말합니다. 복잡도와 학습 능력이 크게 상승합니다.

### 6.1 과적합(Overfitting)과 드롭아웃(Dropout)
* **과적합(Overfitting):** 모델이 너무 복잡해져서 훈련 데이터의 아주 세밀한 노이즈까지 전부 외워버리는 현상입니다. 교안의 그래프에서 점들을 부드럽게 가르는 선이 아니라, 억지로 모든 점을 꼬불꼬불하게 지나가는 파란 선이 과적합 상태를 의미합니다. 이 경우 새로운 데이터에 대한 예측 성능이 뚝 떨어집니다.
* **드롭아웃(Dropout):** 과적합을 해결(Sol)하기 위한 기법입니다. 학습을 진행할 때 네트워크의 특정 유닛(노드)들을 무작위로 선택해 **일시적으로 제거(비활성화)**합니다. 이를 통해 네트워크가 특정 노드에 과도하게 의존하는 것을 방지하고 일반화 성능을 높입니다.

*(참고: 교안 79p ~ 90p)*

## 7. 컴퓨터 비전과 합성곱 신경망(CNN)
컴퓨터 비전(Computer Vision)은 디지털 이미지를 분석하고 이해하는 컴퓨터 모델을 의미합니다. 

### 7.1 이미지의 디지털 표현
* 이미지는 미세한 사각형 격자인 픽셀(Pixel)들로 구성됩니다.
* **흑백(Grayscale) 이미지:** 각 픽셀이 0(완전한 검은색)부터 255(완전한 흰색)까지의 숫자로 명도를 표현하는 하나의 행렬입니다.
* **컬러(RGB) 이미지:** 빨강(R), 초록(G), 파랑(B) 각각이 0~255 값을 가지며, 이 세 장의 행렬(채널)이 겹쳐져 다양한 색상을 만들어냅니다.

### 7.2 이미지 합성곱(Image Convolution)
* 이미지의 각 픽셀과 그 주변 이웃 픽셀들의 값에 **커널 행렬(Kernel Matrix, 필터)**의 가중치를 곱하고 더하는 연산입니다. 이를 통해 모서리(Edge), 선 등의 특징을 추출합니다.
* **연산 예시 (교안 참조):**
    이미지 행렬과 3x3 커널 매트릭스(가운데가 8, 주변이 -1)가 겹쳐지는 부분의 연산
    $$(-1 \times 20) + (-1 \times 20) + (-1 \times 20)$$
    $$+ (-1 \times 50) + (8 \times 50) + (-1 \times 50)$$
    $$+ (-1 \times 50) + (-1 \times 50) + (-1 \times 50)$$
    결과값: $-60 + 400 - 250 = 90$. 이 연산이 전체 이미지를 슬라이딩하며 새로운 행렬(특징 맵)을 생성합니다.

### 7.3 풀링(Pooling)
* **개념:** 입력값의 특정 영역에서 샘플링을 하여 크기를 줄이는(압축하는) 과정입니다. 연산량을 줄이고 중요한 특징만 남깁니다.
* **Max-pooling 예시:** 주어진 $2 \times 2$ 영역 내에서 가장 큰 값만 선택합니다.
    * 좌상단 `[30, 40 / 20, 50]` $\rightarrow$ 최대값 **50**
    * 우상단 `[80, 90 / 100, 110]` $\rightarrow$ 최대값 **110**
    * 좌하단 `[0, 10 / 10, 20]` $\rightarrow$ 최대값 **20**
    * 우하단 `[20, 30 / 40, 30]` $\rightarrow$ 최대값 **40**
    결과적으로 $4 \times 4$ 행렬이 $2 \times 2$ `[[50, 110], [20, 40]]` 행렬로 축소됩니다.

### 7.4 합성곱 신경망(CNN) 전체 아키텍처
1.  **Convolution (특징 추출) $\rightarrow$ Pooling (압축) 연산의 반복:** 앞단에서는 선, 엣지, 모양과 같은 하위 수준의 특징(Low-level features)을 잡고, 층이 깊어질수록 물체 전체의 형태 같은 상위 수준의 특징(High-level features)을 파악합니다.
2.  **Flattening (평탄화):** 2차원 행렬 데이터를 1차원 배열로 쭉 폅니다.
3.  **Fully Connected Network:** 평탄화된 데이터를 바탕으로 최종 이미지가 무엇인지 분류(출력)합니다.

*(참고: 교안 91p ~ 129p)*

## 8. 순환 신경망(RNN, Recurrent Neural Network)
데이터가 시간의 흐름이나 순서를 가질 때 사용하는 신경망 구조입니다.

* **Feed-forward Neural Network:** 입력에서 출력으로 오직 한 방향으로만 신호가 흐르는 구조입니다. (지금까지 살펴본 일반적인 망)
* **Recurrent Neural Network (RNN):** 모델에서 생성된 출력값이 다시 자기 자신의 입력으로 되돌아가는(피드백되는) 루프(Loop) 구조를 가집니다. 과거의 정보가 현재의 처리에 영향을 미치므로 자연어, 음성, 문장 처리 등에 탁월합니다.
* **활용 예시 (Image Captioning):** 교안의 남녀가 걷고 있는 사진 처리 방식입니다.
    1.  **CNN** 모델이 사진 데이터를 입력(Input)받아 시각적 특징을 처리합니다.
    2.  이 특징을 **RNN** 모델에 넘겨줍니다.
    3.  RNN은 루프를 돌면서 단어를 하나씩 차례대로 뱉어냅니다: "group", "of", "people", "walking"... 앞서 생성한 단어가 다음 단어 생성에 힌트(피드백)가 되어 매끄러운 문장(Output)을 완성합니다.

*(참고: 교안 130p ~ 138p)*
