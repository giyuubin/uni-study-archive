- Search -> 출발점, 끝점
 - Agent: 환경을 인식, 그 환경에 행동을 취하는 주체.
 - State: agent와 그것의 환경 구성 상태.
 - **Initial State**: 초기 상태. (1st)
 - **Action**: 가능한 행동. Action(s) (Action -> s(행동))
 - **Transition Model**: 특정 상태에서 행동을 취했을 때 결과로 나타나는 상태에 대한 설명. RESULT(s, a) (s => a => RESULT)
 - State Space <- 자판기, all states
   : the set of **all states** reachable from the **initial state** by any sequence of actions.
 - **Goal Test**: way to determine whether a given state is a goal state.
 - **Path Test**
   
 - Optimal Solution -> all solution
 - node: a data sturcture that keeps track of
   - a state
   - a parent (상위 node 기록)
   - an action
   - a path cost =>(확인) 효율성
 - queue: last-in first-out data type.

 - Informed Search: search strategy that uses problem-specific knowledge to find solutions **more efficiently**
 - Greedy(탐욕, 욕망) best-first search
 - Heuristic(경험적) funtion: 현재 노드에서 목표까지 도달하는데 드는 예상 비용 추정하는 함수.
   Ex) Manhatten distance: 현재 위치와 목적지 사이의 가로, 세로 거리 합을 계산해 목표까지의 거리를 예측한다.
 
 - **A*search** <- optimal <- if(조건)
   : g(n) + h(n)이 가장 낮은 노드를 확장.
   - g(n): 시작-현재 노드까지 도달하는데 걸린 비용.
   - h(n): 현재-목표까지 도달할 것으로 예상되는 **추정** 비용
   - optimal 조건
     - h(n) $<=$ 실제 비용
     - h(n)(현재) $<=$ h(n')(후속) + c(비용)
